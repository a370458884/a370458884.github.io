Linux基础(一)基础

发表于 2017-09-04 | 分类于 linux系统原理

shell

命令解析器：接收用户命令，调用相应的应用程序。

在计算机科学中，shell俗称‘壳’，区别于Kemel(核)。shell分为图形界面shell和命令行shell。各个shell功能差不多，Linux下默认使用bash。

bash

bash 是一个为GNU计划编写的Unix shell。

命令格式

命令 [-options] [参数], --help 查看帮助

常见命令

快捷键

Ctrl+l: 清屏，相当于clear命令；

Ctrl+a：跳转至命令开始处；
Ctrl+e：跳转至命令结尾处；

Ctrl+c：取消命令的执行；

Ctrl+u: 删除命令行首至光村所在处的所有内容；
Ctrl+k: 删除光标所在处至命令行尾部的所有内容；

vi/vim编辑器

一种行编辑器。
VIM有输入模式、编辑模式和末行模式。

模式转换

三种模式。 输入模式，编辑模式(命令模式)，末行模式
vim打开文件时，默认进入的是编辑模式
